# 麦麦解析 (NCMai)

## 简介

一个用于 MaiBot 的插件，支持将 QQ 聊天中接收到的 `.ncm` 加密音乐文件解码为原始音频并自动发送回对话。基于群友 termux 提供的 C 源码迁移为纯 Python 实现，无需外部可执行文件。

## 安装与依赖

### 前置条件
- MaiBot 版本 >= 1.0.0
- 已安装并启用 **SnowLuma** 插件（或其他 OneBot 网关插件）

### 自动安装
插件依赖 `pycryptodome` 和 `aiohttp`，MaiBot 会在加载时自动处理缺失的 Python 包。
