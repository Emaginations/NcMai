# 麦麦解析 (NCMai)

## 简介

这是一个收到QQ聊天中的.ncm文件后会自动解码并发送回去的MaiBot插件。主项目是[maibot](https://github.com/Mai-with-u/MaiBot)。

## 配置

有人性化的webui设计，请在webui插件市场安装并在管理页面进行配置，完成后请重启maibot以安装所需python依赖。
如果你使用docker，请使用或参考[docker-compose(NcMai)](https://github.com/Emaginations/maibot-snowluma_docker-compose/blob/main/docker-compose(NcMai).yaml)进行文件夹暴露，以使snowluma或napcat能够访问到test_ncm_song文件夹。

<br>

### 插件设置
- **开关**：启用/禁用插件。

### 网关设置
- **适配器**：选择 `SnowLuma` 或 `NapCat`，默认 `SnowLuma`。

### 解码设置
- **随机提示语**：在这里添加解码时的提示语。

### 测试命令
- **启用 /ncm 命令**：开关，控制是否允许使用 `/ncm` 测试命令。
- **测试账号**：QQ 号（字符串），若填写则仅该用户可使用 `/ncm` 命令，留空则所有用户均可使用。

<br>

## 命令

| 命令 | 说明 |
|------|------|
| /ncm | 测试解码 `test_ncm_song/伊格赛听 - 逍遥仙.ncm` 文件，并将解码后的音频发送给当前用户（若配置了测试账号则仅限该用户使用）。 |

<br>

## 工作原理

1. **触发时机**：插件通过 **BLOCKING Hook** 拦截所有聊天流中的文件消息，检测是否为 `.ncm` 格式。
2. **随机提示**：根据设定的风格，随机选取一条提示语发送到当前聊天流。
3. **文件下载**：通过文件 URL 将加密文件下载到本地缓存目录。
4. **格式转换**：将 `.ncm` 文件解密还原为原始音频（自动识别 `.mp3`、`.flac`、`.m4a` 等格式）。
5. **文件上传与发送**：调用网关的上传 API 将解码文件上传至 QQ 服务器，获取 `file_id` 后通过 `send.custom` 发送文件消息。
6. **缓存清理**：每次成功发送文件后，自动清空缓存目录（包括本次文件及历史遗留文件）。

<br>

## 测试

- 直接在 QQ 聊天中发送一个 `.ncm` 文件，机器人应返回解码后的音频文件。
- 使用 `/ncm` 命令可测试插件自带的示例文件（需先在 WebUI 中启用该命令）。
