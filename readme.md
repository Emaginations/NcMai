# 麦麦解析 (NCMai)

## 简介

一个用于 MaiBot 的插件，支持将 QQ 聊天中接收到的 `.ncm` 加密音乐文件解码为原始音频并自动发送回对话。本插件仅供学习交流，不内置任何音乐资源。

## 配置

有人性化的webui设计，请在webui插件市场安装并在管理页面进行配置:

<br>

### 插件设置
- **开关**：启用/禁用插件。

### 解码设置
- **随机提示语**：解码时随机发送的提示信息（支持 emoji，最多 21 条），每条不超过 10 字（含 emoji）。

### 缓存清理
- **缓存有效期（秒）**：解码后文件保留时间，默认 86400 秒（24 小时），到期自动清理。
- **按需清理**：开启后，每次收到新文件时主动清理已过期的缓存文件。

<br>

## 安装与依赖

### 前置条件
- MaiBot 版本 >= 1.0.0
- 已安装并启用 **SnowLuma** 插件（或其他 OneBot 网关插件）

### 自动安装
插件依赖 `pycryptodome` 和 `aiohttp`，MaiBot 会在加载时自动处理缺失的 Python 包。

### 手动安装
```bash
pip install pycryptodome aiohttp
